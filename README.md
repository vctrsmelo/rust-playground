# rust-playground
Repository with Rust code implemented for learning purpose.

## Commands

### Create Project
```sh
cargo new <project name>
```

### Format Code
```sh
cargo fmt
```

### Run
```sh
cargo run
```

### Suggestions
Clippy provides hints and guidance while you code. 
```sh
cargo clippy
```

### Update
```sh
rustup check
rustup update
```
